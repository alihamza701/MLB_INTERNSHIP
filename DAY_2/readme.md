## ⚡ Challenges Faced

During the implementation of these programs, I encountered several challenges:

* Understanding the difference between mutable and immutable data types.
* Choosing the appropriate data structure for storing multiple student records.
* Updating and deleting dictionary entries correctly.
* Searching for records efficiently without unnecessary loops.
* Handling user input and converting data into the correct data types.
* Avoiding common dictionary errors such as accessing non-existent keys.
* Managing program flow using functions and menu-driven logic.

---

## ✅ Solutions Implemented

To overcome these challenges, I implemented the following solutions:

* Used dictionaries with **Roll Number** as the unique key for efficient record lookup.
* Broke the program into reusable functions (`add`, `search`, `update`, and `delete`) to improve code organization.
* Used conditional statements to check whether a student record exists before performing operations.
* Improved input handling by converting numeric values using `int()` where required.
* Simplified search, update, and delete operations by checking key existence directly instead of iterating through all records.
* Practiced writing cleaner and more readable code with meaningful variable names.
